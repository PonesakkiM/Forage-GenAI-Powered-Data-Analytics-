## Tasks

### Task 1 — Data Exploration and Quality Assessment

Explored and assessed the quality of raw datasets, identified missing values, outliers, and inconsistencies. Prepared a structured summary of data issues and proposed cleaning strategies.

**Skills demonstrated:** Data profiling · EDA · Data quality frameworks

---

### Task 2 — Predictive Model Plan

Leveraged Generative AI tools to accelerate data analysis, generate hypotheses, and surface patterns that would take significantly longer through manual methods.

**Skills demonstrated:** Prompt engineering · GenAI tooling · Analytical reasoning

---

### Task 3 — Business Summary Report

Built clear, compelling visualizations to communicate findings to a non-technical stakeholder audience. Focused on narrative structure and actionable insights.

**Skills demonstrated:** Data visualization · Business storytelling · Report writing

---

### Task 4 — AI Collections System

Synthesized all prior findings into a concise executive-level report with strategic recommendations backed by data evidence.

**Skills demonstrated:** Executive communication · Strategic thinking · Report writing
